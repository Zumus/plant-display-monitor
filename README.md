# Plant Monitor

Code for monitoring soil moisture on a coffee plant.
Currently uses the following hardware:
- Waveshare [ESP32-S3-Touch-LCD-2](https://www.waveshare.com/product/arduino/boards-kits/esp32-s3/esp32-s3-touch-lcd-2.htm?___SID=U)
- Generic Capacitive Soil Moisure Sensor v1.2

Using this as a project to tinker with ESP32 display boards and learn a bit about the (LVGL)[https://lvgl.io/docs/open] library for graphics.
